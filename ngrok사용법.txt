** 간단사용법
ngrok.exe를 실행한 후 아래 붙여넣기 하면 github 양방향 동기화가 됨.
ngrok http 80


** 셋팅방법
ngrok 설치:
https://ngrok.com 에서 회원가입 후 다운로드
다운로드 받은 ngrok.exe를 편한 위치에 저장

ngrok 웹사이트의 대시보드에서 Your Authtoken을 복사하여 다음 명령어를 실행해주세요:
ngrok config add-authtoken 당신의_인증토큰

토큰을 복사한 후:
명령 프롬프트 창에 ngrok config add-authtoken 입력하고
뒤에 복사한 토큰을 붙여넣기
예시: ngrok config add-authtoken 2MS9xxxxxxxxxxxxx

인증이 완료되면 다음 단계로:

XAMPP Control Panel에서 Apache가 실행 중인지 확인하고
명령 프롬프트에 ngrok http 80 을 입력해주세요

이제 GitHub 웹훅을 설정하겠습니다:

GitHub 저장소로 이동
Settings → Webhooks → Add webhook 클릭
다음 정보를 입력:
Payload URL: cmd 터미널 창에 나오는 Forwarding의 https://주소복사/webhook.php
Content type: application/json
Secret: smart-farm-hub-secret (webhook.php 파일에 설정한 값)
SSL verification: Disable (체크 해제)
Just the push event 선택


** 내 로컬에 있어야 하는 파일
webhook.php
$secret = "아무거나 16자이상"; // GitHub에서 설정할 시크릿키
